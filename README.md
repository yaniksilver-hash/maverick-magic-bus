# The Maverick Magic Bus 🚌🌈

A standalone, interactive "voyage" experiment for Maverick1000 — a hand-drawn green
Magic Bus you can choose to board. Built as a single static `index.html` (no build step).

- **Yes!** — confetti, a horn toot, and a stamped boarding pass.
- **No.** — the bus loops back around anyway. 😏
- **???** — a "mystery seat" slot machine of Maverick archetypes.

Art: Yanik's original hand-drawn doodle, recolored to Maverick green.
Experiment only — copy lives in the Maverick working drive.
